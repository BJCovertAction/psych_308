ANCOVA

#Ancova code. Note that it also has arguments for checking homogeneity of variance (Levene's) and Normality (Shapiro Wilks Test) as well as a qq plot for visualizing multivariate normality

model.ancova <- ancova(dat.anxiety, 
                       dep = 'posttest', #dependent variable
                       factors = c('group'), #independent variable
                       covs = c('pretest'), #covariate (control variable)
                       effectSize = 'partEta', #effect size
                       homo = TRUE, #homogeneity of variance (Levene's Test) results
                       norm = TRUE, #nomality (Shapiro Wilks Test) - tests if your data differs significantly from perfectly normal data
                       qq = TRUE,  #returns a qq plot
                       postHocES = 'd', 
                       postHoc = c('group'), #this is the variable 
                       postHocCorr = 'tukey', #post-hoc correction
                       emMeans = list('group'), #generate the adjusted means
                       emmTables = TRUE) #put the adjusted means in a table

Post Hocs with CIs and Effect Sizes

# Fit ANCOVA model using lm
ancova_model <- lm(posttest ~ group + pretest, data = dat.anxiety)

# Perform Tukey-adjusted pairwise comparisons
posthoc_results <- summary(glht(ancova_model, linfct = mcp(group = "Tukey")))

# Extract the mean differences, standard errors, and confidence intervals
mean_diffs <- posthoc_results$test$coefficients
se_diffs <- posthoc_results$test$sigma
ci <- confint(posthoc_results)$confint

# Calculate adjusted means for each group with covariate held at mean
adjusted_means <- predict(ancova_model, newdata = dat.anxiety %>% 
                            group_by(group) %>% 
                            summarize(pretest = mean(pretest)))

# Calculate Cohen's d using standard deviation of adjusted means
cohens_d_values <- mean_diffs / sd(adjusted_means)

# Compile results into a data frame
results_df <- data.frame(
  Comparison = names(mean_diffs),
  Mean_Difference = mean_diffs,
  SE = se_diffs,
  Lower_CI = ci[, "lwr"],
  Upper_CI = ci[, "upr"],
  Cohen_d = cohens_d_values
)

# Display the results
print(results_df)

Lay Write Up Considering peoples baseline anxiety scores, the more intense exercise people perform, the lower their anxiety. Indidivudals looking to reduce their anxiety should perform high intensity exercise.

Journal (APA) Write Up The present study tested the relationship between exercise intensity level and anxiety. Forty-five participants were randomly assigned to three exercise intensity levels (low, medium, high) Anxiety levels were measured at baseline and after six months of the exercise program. We specified an ANCOVA model to predict posttest anxiety based on exercise intensity group while controlling for pretest anxiety scores.

Prior to model specification, we confirmed the data met all assumptions of ANCOVA. A histogram of model residuals appeared normally distributed and skew and kurtosis levels of posttest anxiety were within 3 and 10 respectively for each exercise group. A Levenes test confirmed the data met the homogeneity of variance assumption, F(2,42) = 0.26, p = .772. Scatterplots and correlations coefficient confirmed both a linear relationship between the covariate (pretest anxiety) and posttest anxiety, as well as homogeneity of regression amongst the groups. Finally, there was no significant relationship between the covariate (prettest) and the independent variable (exercise intensity), F(2,42) = 0.37, p = .693, \(\eta\)2 = .02.

The ANCOVA model confirmed a significant difference in posttest anxiety based on exercise intensity group, F(2,41) = 151.26, p < .001, \(\eta\)2 = .88 as well as a significant relationship between pretest and posttest anxiety, F(1,41) = 404.29, p < .001, \(\eta\)2 = .91. There was a large effect of exercise intensity on post-test anxiety, while controlling for baseline anxiety levels, exercise intensity explained 91% of the variance in posttest anxiety levels.

Post-hoc testing using the Tukeys procedure indicated individuals who did moderate exercise (Madj = 15. 79) for six months experience significantly less anxiety than those who performed light exercise (Madj = 16.34), Mdiff = 0.64, t(41) = 4.24, p < .001. Additionally, individuals who did high intensity exercise (Madj = 13.46) experienced lower anxiety than those who performed moderate exercise, Mdiff = 2.34, t(41) = 15.52, p < .001. Overall, when controlling for baseline anxiety scores, the more intense the exercise program, the lower ones anxiety level.
